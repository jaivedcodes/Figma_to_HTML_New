Figma → HTML Generator

A production-ready CLI tool that converts Figma designs into pixel-perfect, responsive, Bootstrap-based HTML pages — automatically.

1
Get your Figma Personal Access Token Figma → Settings → Personal Access Tokens → Create new token
2
Install dependencies (first time only) npm install
3
Run the generator npm run generate The CLI will ask for your Desktop Figma URL, Mobile Figma URL, API token, and project name.
4
Open your generated site output/<your-project-name>/index.html
Tech stack: Node.js · TypeScript · Bootstrap 5.3.3 · Figma REST API · Sharp · SVGO · Prettier
